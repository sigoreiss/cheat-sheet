HTML Tables

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of rows containing actual table data
<tfoot> Table Foot The set of rows defining the footer in a table
<tr> Table Row The table row container
<td> Table Data The table cell container
<th> Table Header Defines a cell as header of a group of table cells
Attributes
colspan Defines how many columns a <td> element should span
rowspan Defines how many rows a <td> element should span
scope Defines the cells that the <th> element relates to

CSS Selectors & Specificity

Each subsequent type of a CSS selector is more specific than the previous one

tag selector

Selects all HTML tags of the specified type

.class selector

Selects all HTML tags with the specified value of the class attribute

#id selector

Selects all HTML tags with the specified value of the id attribute

Note: If you type !important after any CSS declaration, it will give it maximum possible specificity. Use sparingly and very carefully!
